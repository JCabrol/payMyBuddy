# payMyBuddy

A transfer money application permitting to send easily money to other person connected.

## Technical specifications

PayMyBuddy version 0.0.1-SNAPSHOT uses 
* Java 13
* Maven 4.0.0
* Spring Boot 2.6.3
* MySql 8.0
